Create Ethereum wallet on myetherwallet.com

1. Go to https://myetherwallet.com. And follow the instructions to create your wallet ether.

* In case you are using Exodus wallet - you don’t need to create the address. Use the private key from the Exodus wallet to access your Ethereum address via myetherwallet.com:

— Export a private key: if you are on macOS, the develop menu is under 'Exodus'. If you are using Windows or Linux, it can be accessed by pressing Ctrl+Shift+D.

Select Developer > Assets > Ethereum > Export private keys

Go to the folder and open the export file. You will see csv file - open it and find you private key (the sybmols after the last comma), copy it.

— Then go to https://myetherwallet.com and select Send Ether & Tokens. Enter your wallet using the Private Key

2. Save your file and click the button that you understand. Continue.

3. On the next screen save your private key and click on the save your address button.

4. On the next screen choose How do you want to access your wallet. Select the Keystore File (UTC / JSON) and click on the button on the right. Select the file that you saved in step 2 and enter the password for access.

5. After unlocking the address you will see below your wallet number.
